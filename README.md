how to use
----------

* npm install

* bower install

* start mongoDB

    *.* on windows  
	```
	mongod --config ./mongod.conf  
	```
	*exe and conf are located in C:\Program Files\MongoDB\Server\3.0\bin\*
	```
	mongod.conf  
		bind_ip = 127.0.0.1

		dbpath = C:\mongodb\data\db

		logpath = C:\mongodb\log\mongo-server.log
	```

    *.* on Mac  
	```
	sudo mongod
	```

* npm start
